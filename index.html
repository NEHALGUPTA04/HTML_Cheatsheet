  • Headings:

    There are six headings available in HTML, H1 is the largest and H6 is the smallest among all.


    1. Heading 1: <h1> tag
      <h1>Heading 1</h1>
    2. Heading 2: <h2> tag
      <h2>Heading 2</h2>
    3. Heading 3: <h3> tag
      <h3>Heading 3</h3>
    4. Heading 4: <h4> tag
      <h4>Heading 4</h4>
    5. Heading 5: <h5> tag

  • div tag:
    div tag or division tag is used to make blocks or divisions in the document. 
    <div>This is div block </div>
  • span tag:
    span is a container for inline content 
    <span>This is span block </span>
  • p tag:
    Parageaph 
    <p>This is paragraph </p>
  • pre tag:
    pre tag represents pre-formatted text 
    <pre> Nehal Gupta </pre>
  • code tag:
    code tag is used to represent source codes 
    <code>import python </code>
  • Text Formatting:

    Text formatting tags are used to format text or data of HTML documents. You can do certain things like creating italic, bold, strong text to make your document look more attractive and understandable.


    1. <b> tag
      <b>I'm bold text</b>
    2. <strong> tag
      <strong>I'm important text</strong>
    3. <i> tag
      <i>I'm italic text</i>
    4. <sub> tag
      <em>Emphasized text</em>
    5. <sup> tag
      <sub>Subscript</sub>
